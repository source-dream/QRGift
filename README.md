## 设计理念

现如今，个人定制双铜纸彩印红包价格偏贵且定制的效果没有那么好，不如直接用一个卡片来当作红包，即二维码红包。其实之前出现过这种类型的红包，但是安全性是一个很大的问题，可能有人恶意领取，而且支付宝红包是有时间限制的，这个时候就需要自己部署一个管理平台来管理自己打印的电子二维码，这个项目正是为了解决这个问题而生的！
## 项目介绍

批量生成唯一的红包二维码图片，管理后台可以对生成的二维码进行红包管理，绑定真实的红包领取链接到该二维码，以此实现安全可靠的电子红包。
